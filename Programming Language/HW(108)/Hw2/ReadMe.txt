Prob1_1 : 
	function : (prime n)
	直接呼叫(prime n)便可以判斷是否為prime
	script裡有做2,239,999,17的範例
Prob1_2 : 
	function : (palindrome '(string))
	直接呼叫(palindrome '(string))便可以判斷是否為prime
	script裡有做'(a b c),'(m a d a m),'(cat dog),'(),'(cat dog bird bird dog cat)的範例
Prob1_3 : 
	function : (fib1 n),(fib2 n)
	呼叫(fib1 n)可以做原始版本的fibonacci，呼叫(fib2 n)則是做tail recursion的fibonacci
	script裡有做(fib1 8),(fib2 8)的範例，並且有加trace
Prob2 : 
	function : (mergesort list)
	首先先輸入要list的數量，然後再輸入list的資料(用空白間隔)，便會求出答案並印出來
